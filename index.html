7 Tipos de Linux para voce Aprender

O linux é um ótimo sistema operacional, ele contém tudo o que é necessário para se trabalhar com um sistema operacional estável e gratuito com uma grande comunidade envolvida na sua manutenção e melhorias constantes.

Essse site tem por objetivo exemplificar 7 versões de linux nas quais os usuários de outros sistemas operacionais tenham vontade de migrar para testar ou ate adotar o linux como seu sistema padrão.


Ubuntu

O linux Ubuntu é um ótimo sistema de desktop para um usuário que está iniciando nesse mundo interessante do open source, além de poder utilizar o esse sistema como Desktop em seu site oficial possui também a sua versão pra servidor, o Ubuntu Server, muito utilizada por empresas devido a segurança por ter uma vasta comunidade por trás trabalhando mais rapidamente para identificar bugs e corrigi-los, além de não ser necessário licensa nem custo algum para poder utilizar esse sistema em um servidor.

Para Usuários comuns de Desktop, ele se faz muito bom pois possui uma interface muito amigável e bem facilitado em seu uso. Seu menu de configuração se assemelha muito ao do celular, e possui uma loja onde é possível fazer download de alguns programas bem legais para videos, edição de imagens e logomarcas como por exemplo o inkscape e o gimp.

O Ubuntu também permite ser instalado facilmente ao lado de outro sistema operacional caso você não queira migrar de vez de seu sistema operacional para ele, você pode fazer o chamado dual boot que é manter os dois sistemas operacionais no mesmo hd porém em partições diferentes. O próprio Ubuntu cuida de fazer esse procedimento de forma automatizada, só basta colocar em um pendrive bootável, rodar ele e ter uma ótima experiência de uso.

Pode ser um contra utilizar em dual boot pois como seu sistema mais usual vai estar instalado e você vai ter liberdade pra escolher um dos dois, pode haver um impulso de você deixa-lo um pouco de lado para utilizar o seu sistema mais usual. Mesmo assim invista em um tempo com o ubuntu, ou qualquer outra distribuição de sua preferencia, quando você acostumar, terá experiências incríveis.

Linux Mint

Se você gostaria de ter uma experiencia em linux com uma interface mais amigável e mais parecida com o sistema operacional windows, você pode optar pela instalação do Linux mint.

O Mint é outra ótima distribuição linux para quem está queredo experimentar esse sistema, ele é baseado em debian assim como o Ubuntu porém a sua interface gráfica é diferente do anterior, é mais parecida com aquele padrão do windows, onde se tem uma barra de tarefas que ficam com as aplicações abertas, sua interface gráfica é conhecida como Cinammon.

O processo de instalação é praticamente parecido com o do Ubuntu e o usuário consegue ter uma experência muito agradável durante a instalação pois ela é bastante auto intuitiva.

POP OS

Outra Distro linux bem interessante para usuários e Desktops, e com um design bem interessante ambiente gráfico Gnome mas com algumas peculiaridades bem legais é o POP OS.

A loja de aplicativos desse Sisetma é é muito boa, o próprio atualiador do sistema é gerenciador pela loja de aplicativos dele, chamada de POP Shop essa loja é derivada de um outro projeto do Elementary Os que também é outro sistema operacional linux e possui uma loja muito bonita e bem elaborada.

O POP OS teve como um dos motivos de grande crescimento o projeto steam play da Valve que consegue rodar alguns jogos de windows no linux, isso tem chamado a atenção de vários gamers que gostariam de experimentar algo novo em experiência de sistema operacional mas sem perder a alegria quese tem dos games que outros sistemas operacionais tem mais facilidade de instalação e acesso.

MX Linux

Baseado em Debian, esse sistema possui uma interface simples mais muito amigável e bonita na minha opinião, ele possui ambientes gráficos como o xfce que é um abiente bem simples mas muito poderoso. Esse sistema é ótimo para computadores que tenham pouco processamento pois ele deixa os computadores super leves e rápidos.

Pude ter uma ótima experiência utilizando essa distro no meu antigo notbook, um amd e300 e o notbook parecia outro totalmente diferente, a velocidade na qual os programas eram abertos e a praticidade que tinha com ótimos programas que já vieram pre-instalados fez essa experiência tornar o MX Linux um dos meus queridinhos. Atualmente utilizo como distro principal em meu pc.

Essa pequena experiência trás uma idéia de como é ter o MX linux como sistema principal para Desktop, além de funcionar muito bem em computadores com pouco processamento e configurações baixas, ele possui vários progrmas que facilitam bastante o uso do dia dia. O sistema trás informações de atualizações muito facilmente durante o uso, sua loja de aplicativos não é tão amigável para quemesá iniciando no linux pois é o synaptic, porém mesmo não tendo aquela beleza gráfica dos sistemas citados anteriormente, possui uma vasta quantidade de aplicativos e estáveis como por exemplo o virtual box, vlc player, entre outros.

Para Desenvolvedores, ele funciona super bem, tive experiencias de instalação com as iDEs tanto do VSCode como do Eclipse IDE e ele funcionou muito bem, além do banco de dados Postgres SQL no qual tive uma experiencia muito boa de uso.

Para usuários com dificuldade em utilizar o Synaptic, pode-se instalar o Gnome-store que ele também funciona corretamente.

Deepin Linux

O Deepin Linux é uma distribuição de origem chinesa baseada em Debian e ótima para ambiente de Desktop, principalmente para aqueles usuários que gostam de uma boa funcionalidade mas com toda aquela beleza de temas por trás.

Se você se identifica com um sistema “Bonitinho” esse sistema é super indicado para você. Ele possui papeis deparede belíssimos e formas de optimização da área de trabalhos que o deixam super embelezado. A loja do Deepin linux é uma das mais completas com bastante aplicativos disponíveis para instalação, além disso pe uma das lojas mais bonitas se não for considerada a mais bela dentre as distros linux. O sistema é bem prático e simples de utilizar e desperta bastante alegria por parte dos usuários que a utilizam pois torna toda a experiência de uso desse usuário mais bonita.

Parrot Linux

Se interessa por pentest, hacker e técnicas de invasão? Então você deveria experimentar o parrot linux. Ele conta com várias aplicações de hacker e pentest para o usuário praticar técnicas, dentro de um ambiente controlado é claro.

Como informado na descrição esse sistema não é muito indicado para usuários comuns de Desktops, pois como ele é voltado para testes de invasão, a parte mais amigável do sistema operacional se perde um pouco. Para poder utiliza-lo como sistema principal, deve-se ter um pouco de conhecimento mais aprofundado em infromática, pois ferramentas dele favorecem muito mais essas técnicas, do que por exemplo ferramentas de editor de foto, comuns por exemplo em sistemas linux mais amigavéis para os usuários comuns. Porém é super inidicado para estudantes de segurança da informações pois possui ferramentas muito importantes para essa área de extremo valor na informática e que depende de mais profissionais a cada dia devido ao aumento da criminalidade cibernética.
